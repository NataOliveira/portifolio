## 📱 Sobre o Projeto

Este é o meu **portfólio profissional**, desenvolvido para centralizar minhas principais criações, expor meu conjunto de habilidades técnicas e demonstrar minha evolução como desenvolvedor. O site conta com uma interface moderna em **Dark Mode**, fluida e totalmente responsiva, projetada para fornecer uma excelente experiência de usuário (UX).

---

## 🚀 Tecnologias e Ferramentas

O projeto foi construído utilizando práticas modernas de desenvolvimento web front-end:

*   **HTML5** – Estruturação semântica e acessível.
*   **CSS3** – Estilização modularizada, uso avançado de variáveis nativas (`:root`) e efeitos visuais.
*   **Spline 3D Integration** – Incorporação de elementos tridimensionais interativos na seção Hero via `@splinetool/viewer`.
*   **Font Awesome** – Ícones vetoriais modernos.
*   **Mobile First & Responsividade** – Adaptação fluida para dispositivos móveis, tablets e desktops de alta resolução.

## 🎨 Características Técnicas e UX

Analisando os bastidores do código, o projeto se destaca por:

*   **Arquitetura CSS Organizada:** Divisão modular de estilos através da importação de arquivos específicos (`header.css`, `hero.css`, `cards.css`, `rodape.css`, `media.css`), facilitando a manutenção.
*   **Identidade Visual Forte:** Paleta de cores escura com gradientes modernos e vibrantes (`#2C88C4` a `#45DD3E`), criando alto contraste e legibilidade.
*   **Efeitos Modernos:** Uso de `backdrop-filter: blur(10px)` (Efeito Glassmorphism) e sombras estratégicas para criar profundidade nos cards de habilidades.
*   **Navegação Fluida:** Utilização de `scroll-behavior: smooth` para transições suaves entre as seções da página através do menu de navegação.
*   **Microinterações:** Efeitos de *hover* refinados com propriedades de `transition` e `transform: scale()`, que aumentam o engajamento visual sem prejudicar o desempenho.
